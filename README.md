# 🔍 Detecção de Anomalias em Transações Financeiras

## 📖 Sobre

Este projeto foi desenvolvido como desafio prático da DIO com o objetivo de identificar transações financeiras consideradas suspeitas utilizando Python.

A aplicação lê um conjunto de dados contendo transações, analisa os valores e identifica automaticamente operações que ultrapassam um limite definido como suspeitas.

---

## 🚀 Tecnologias utilizadas

- Python
- Pandas

---

## 📂 Estrutura do projeto

- `main.py` → Código principal
- `dados.csv` → Base de dados das transações
- `requirements.txt` → Dependências do projeto
